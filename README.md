🖊️ Текстовый редактор
В форуме и архиве задач реализован WYSIWYG редактор с поддержкой:

Функция	Описание
B I U S	Жирный, Курсив, Подчеркнутый, Зачеркнутый
Выравнивание	По левому краю, по центру, по правому краю
Шрифты	Arial, Times New Roman, Courier New, Georgia, Verdana, Comic Sans MS
Размеры	Маленький, Средний, Большой, Очень большой
Цвета	Цвет текста и фона (color picker)
Кавычки	«», "", „“
Счетчик	Отображение количества символов
Перенос	Автоматический перенос длинных слов


📦 Хранение данных (замена бекенда)
Бекенд отсутствует, данные хранятся в localStorage:

Компонент	Где хранится
Пользователи	localStorage.users
Текущий пользователь	localStorage.currentUser
Данные задач	Статические данные в pages/archive/[id].vue
Данные форума	Статические данные в pages/forum/[id].vue
