# 🏥⚕️ Medical School


## 📋 О проекте
**Medical School** — это демо-приложение на Vue.js.


## 🚀 Основные функции

### 🔐 Авторизация и профиль
- Регистрация с сохранением данных в `localStorage`
- Вход/выход из аккаунта
- Редактирование профиля (ФИО, отчество)
- Защита маршрутов (Vue Router guards) с редиректом для неавторизованных пользователей

### 📚 Архив задач
- Таблица с медицинскими задачами по дисциплинам
- 🔍 Поиск по названию, дисциплине или ID
- 🏷️ Фильтрация по дисциплине
- 🔽 Сортировка по клику на заголовки таблицы
- 📝 Страница задачи с условием и вариантами ответов

### 💬 Форум
- Список тем с категориями
- 🔍 Поиск по темам, авторам или содержанию
- 🏷️ Фильтрация по категориям (Учеба, Медицина, Олимпиады, Обсуждения, Вопросы)
- 🔽 Сортировка: новые, старые, по популярности
- 💬 Страница темы с ответами (вложенность с отступами)

### 📊 Состояние системы
- Таблица со статусами проверки решений
- 🔍 Поиск по ID, задаче или автору
- 🏷️ Фильтрация по статусу (Принято, Неверный ответ)
- 🟢🔴 Цветовая индикация статусов

## 🖊️ Встроенный WYSIWYG редактор
Реализован в форуме и архиве задач с поддержкой:

| Функция | Описание |
| :--- | :--- |
| **B I U S** | Жирный, Курсив, Подчеркнутый, Зачеркнутый |
| **Выравнивание** | По левому краю, по центру, по правому краю |
| **Шрифты** | Arial, Times New Roman, Courier New, Georgia, Verdana, Comic Sans MS |
| **Размеры** | Маленький, Средний, Большой, Очень большой |
| **Цвета** | Цвет текста и фона (color picker) |
| **Кавычки** | Быстрая вставка: «», "", „“ |
| **Счетчик** | Отображение количества символов в реальном времени |
| **Перенос** | Автоматический перенос длинных слов |

## 📦 Архитектура и хранение данных
Бэкенд отсутствует. Данные хранятся локально в браузере:

| Компонент | Где хранится |
| :--- | :--- |
| Пользователи | `localStorage.users` |
| Текущий пользователь | `localStorage.currentUser` |
| Данные задач | Статические данные в `pages/archive/[id].vue` |
| Данные форума | Статические данные в `pages/forum/[id].vue` |
